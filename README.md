# File Sharing App 🚀

An awesome file-sharing app using Next.js, Tailwind CSS, React, and Firebase! 🌐💾

## Features ✨

- **Easy Upload**: Drag-and-drop files for quick sharing. 📁✨
- **Share Anywhere**: Generate links or send emails for sharing. 📬🔗✉️

## Tech Stack 🛠️

- **Next.js** 🚀
- **Tailwind CSS** 🎨
- **React** ⚛️
- **Firebase** 🔥
